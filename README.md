# CampusHub - 校园互助服务平台

软件工程与计算II 课程大作业

## 项目介绍
CampusHub 是一个面向校园场景的一站式互助服务平台，旨在为学生提供便捷、安全、高效的互助服务。
平台包含：快递代取、学习辅导、二手交易、活动组队、物品借用等功能。

## 项目阶段
- P0：项目启动与AI协作契约
- P1：需求分析
- P2：系统设计
- P3：编码实现
- P4：测试与部署

## 技术栈（暂定）
- 前端：HTML/CSS/JavaScript / Vue
- 后端：Java + Spring Boot
- 数据库：MySQL
- 版本控制：Git + GitHub
